# painting-2d-shapes
Лабораторная работа 11 по компьютерной графике на построение различных 2D фигур и их плоское/градиентное закрашивание.

## Работа с проектом
После запуска программы, можно менять способы закрашивания, кликая на цифры на клавиатуре:
- 2: плоское закрашивание - цвет задаётся константой в шейдере
- 3: плоское закрашивание - цвет передаётся во фрагментный шейдер через uniform-переменные из основной программы
- 4: градиентное закрашивание (каждая вершина своим цветом)

## Стек
Мы используем:
- SFML **2.6.1** (!!!) 
- C++17
